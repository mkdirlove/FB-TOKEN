# Facebook_Token
Grab Your facebook token to activate guard profiles.
# Instalation
```bash
$ apt update && apt upgrade
$ apt install git
$ pkg update && pkg upgrade
$ pkg install python2
$ git clone https://github.com/ITermSec/Facebook_Token.git
$ cd Facebook_Token
$ pip2 install -r requirements.txt
$ python2 install.py

Done!.
```
